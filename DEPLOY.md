# Offscreen Kids Club: Facebook Launch Deploy

This package is ready to publish as a static landing page.

## Goal

Have a clean public URL to post in Dads in Prague so this feels like a real club, not another WhatsApp group.

## Fastest path

1. Create a new GitHub repository:
   - Name: `offscreen-kids-club`
   - Owner: `niclunic`
   - Visibility: Public or Private
   - Do not add README, .gitignore or license

2. Upload these files to the repository root:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `vercel.json`
   - `og-image.png`
   - `README.md`

3. Go to Vercel.
4. Click `New Project`.
5. Import `niclunic/offscreen-kids-club`.
6. Framework preset: `Other`.
7. Build command: leave empty.
8. Output directory: leave empty.
9. Click `Deploy`.
